# Blaze プラン 月額コストの目安

現在の実装（Firestore + Cloud Functions 2本 + Auth）での**おおよその月額**です。  
利用量が少ないほど無料枠内に収まり、**0円**になる可能性が高いです。

---

## 無料枠（Blaze でもそのまま使える）

| 項目 | 無料枠 |
|------|--------|
| **Cloud Functions** | 200万回/月、40万 GB秒、20万 CPU秒、送信 5GB/月 |
| **Firestore 読み取り** | 5万回/日 |
| **Firestore 書き込み** | 2万回/日 |
| **Firestore 保存** | 1 GiB |
| **Authentication** | 月間アクティブ 5万人まで無料 |

※Blaze は「Spark の無料枠 + それを超えた分だけ課金」です。

---

## このアプリの想定利用

- **マッチング**: 1バトルあたり 部屋検索・参加/作成・準備・詩送信・終了 で Firestore の読み書きが複数回
- **画数資産**: ログイン時・入場料・詩送信・課金完了時に `users/{uid}` の読み書き
- **Cloud Functions**: 「購入」クリックで `createCheckoutSession`、決済完了で `stripeWebhook` が 1回ずつ

---

## 規模別の目安（日本リージョン想定・税別）

### 小規模（月間 〜数百バトル・課金 〜数十回）

- **Functions**: 呼び出し 数百〜数千回/月 → 無料枠 200万回以内 → **0円**
- **Firestore**: 読み 数千〜1万/日、書き 数千/日 → 無料枠内 → **0円**
- **合計**: **0円/月** になることが多いです。

### 中規模（月間 数千バトル・課金 数百回）

- **Functions**: 数千〜1万回/月 → 無料枠内 → **0円**
- **Firestore**: 読み 2万/日・書き 1万/日 程度までなら無料枠内 → **0円**
- それを超えると超過分のみ課金（例: 読み 10万回/日 超過で 約 $0.04/日 ≒ 数円/日）
- **合計**: **0円〜数百円/月** 程度のイメージです。

### 大規模（月間 数万バトル・課金 数千回以上）

- **Functions**: 100万回/月 でも 約 $0.40（約 60円）程度
- **Firestore**: 読み 50万/日 超過分 約 $0.19/日、書き 20万/日 超過分 約 $2.30/日 など（リージョンにより変動）
- **合計**: **数千円〜1万円/月** になる可能性があります（使い方次第）。

---

## まとめ

| 利用規模 | 月額の目安 |
|----------|-------------|
| 個人・少人数で運用 | **0円**（無料枠内） |
| 月数百〜数千 PV・課金も少なめ | **0円〜数百円** |
| 月数万 PV・課金も多い | **数百円〜数千円**（場合により 1万円前後） |

- まずは **予算アラート**（例: 500円 or 1,000円で通知）を Google Cloud コンソールの「予算とアラート」で設定しておくと安心です。
- 無料枠を超えても、**超過した分だけ**従量課金なので、小〜中規模なら月額はあまり大きくならない想定です。

※料金は Firebase / Google Cloud の公式料金表に依存し、為替・税率で変動します。正確な金額は [Firebase 料金](https://firebase.google.com/pricing?hl=ja) と [Firestore 料金](https://cloud.google.com/firestore/pricing?hl=ja) で確認してください。
